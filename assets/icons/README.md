# Cause Icons

The dropdown now loads icons from this folder (not from JS emoji text).

Replace these files with your preferred images while keeping the same names:

- `favicon.svg` (in this folder; used as the default search icon)
- `disaster-relief.svg`
- `medical-aid.svg`
- `poverty-relief.png`
- `education.svg`
- `child-welfare.png`
- `disability-support.svg`
- `elder-care.png`
- `community-development.png`
- `environment.png`
- `animal-welfare.png`

Notes:

- You can use `.svg` or `.png`, but if you change extension update paths in `script.js`.
- Recommended size: `64x64` or `128x128` with transparent background.
